\label{reverse()} Write a \textbf{recursive function} that reverses a string (e.g. ``Racecar'' yields ``racecaR'').

\begin{answer}
\begin{lstlisting}
def reverse( string ):
    if string == '' :
        return string
    else:
        return reverse( string[1:] ) + string[0]
\end{lstlisting}
Other solutions are possible.
\end{answer}
